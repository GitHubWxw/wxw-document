### 一、基本语法

#### （1）常用语句

- DATEDIFF() 函数返回两个日期之间的时间

  ```sql
  DATEDIFF(datepart,startdate,enddate)
  ```

- Cast(字段名 as 转换的类型 )  进行SQL 层面类型转换

  ```sql
  转换的类型：
  CHAR[(N)] 字符型
  DATE 日期型
  DATETIME 日期和时间型
  DECIMAL float型
  SIGNED int
  TIME 时间型
  ```
