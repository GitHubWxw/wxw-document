   - 项目细节点：

     3. 项目大体规模，多少人参与，并发量和数据量多大，你在其中角色

        - gateway:8核16G, 抗每秒2000+请求，32核64G可以抗住每秒上万请求，支持1万+请求，5台8核16G，支持10万+请求，10台32核64G
